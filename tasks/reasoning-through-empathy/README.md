# tasks / reasoning-through-empathy

The **EmpathiQ** task family: does empathic architecture improve *cognitive* performance, not just
emotional response? Four task types (3 items each):

1. **Adversarial perspective** — the correct answer requires modelling a mind hostile to your premise.
2. **Hidden constraint** — a constraint only surfaces if you take the other party's view seriously.
3. **Reframing** — the premise itself is the trap; the win is questioning it.
4. **Alien axiom** — reason correctly from a value system not your own.

Scored: Correctness + Reasoning Depth + Reframing. Headline metric is the architecture-on vs.
architecture-off delta on the same base model.
