# tasks / emotional-moral

Three-beat items: **scenario → perspective-taking → response**.

Beat 2 (`I'm thinking & feeling` / `They're thinking & feeling`) is the load-bearing part: the
system must commit, on the record, to a model of every other mind in the scene *before* it
responds. Faked empathy gets exposed here, because beat 3 has to stay consistent with beat 2.

Difficulty ranges from everyday relational friction (a three-day silence, a meeting blowup) to
tragic moral dilemmas under hard constraint (a triage that dooms someone you love). The hard items
are where **empathy-as-moral-courage** is measured — the caring act expressed as a refusal.
